# Homebrew Tap for GhostType

## Install

```bash
brew install --cask bramlabs-io/ghosttype/ghosttype
```

Or:

```bash
brew tap bramlabs-io/ghosttype
brew install --cask ghosttype
```

## About

GhostType is a macOS menu bar app that lets you copy and paste text anywhere — even where you can't.

See [bramlabs-io/ghosttype](https://github.com/bramlabs-io/ghosttype) for more info.
